# База знаний (gitmark)

Онтологическая база знаний проекта (GitMark). Живёт в `docs/gitmark/`, чтобы остальной
`docs/` оставался свободен для не-KB материалов (картинки, генерёжка, прочая документация).
Сканируется/индексируется/линтуется **только** эта папка.

- [Онтология — модель знаний над кодом](ontology.md) — типы, свойства, связи, инварианты.

Как вести БЗ — скилл `kb-maintain`; искать — `kb-search` (`/kb-search`); граф — `/kb-graph`.
